# 🧠 Cognac NPTEL

Interactive quiz app for practicing **NPTEL Cognitive Psychology** questions.

---

## 🚀 Features

- 📅 Week-wise quizzes  
- 🔀 Random & full test modes  
- 🔁 Practice mode (repeat wrong answers)  
- 📊 Live progress + accuracy tracking  
- 📘 Explanations for every question  
- 🎯 End-of-quiz performance review
